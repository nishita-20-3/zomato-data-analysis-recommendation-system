# Zomato Data Analysis & Recommendation System

## 📌 Overview
This project analyzes the Zomato Bangalore restaurant dataset and builds a restaurant recommendation system using Python and Machine Learning techniques.

The project includes:
- Data Cleaning
- Missing Value Handling
- Outlier Detection
- Exploratory Data Analysis (EDA)
- Data Visualization
- K-Means Clustering
- Restaurant Recommendation System

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset
[Zomato Bangalore Dataset](https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants)

---

## 📈 Key Features
- Analyzed 50K+ restaurant records
- Performed statistical analysis using skewness and kurtosis
- Visualized restaurant trends and correlations
- Applied K-Means clustering for restaurant segmentation
- Built recommendation system based on:
  - Cuisine
  - Location
  - Budget

---

## 🤖 Recommendation System Workflow
1. Filter restaurants based on user preferences
2. Identify similar restaurants using clustering
3. Rank restaurants using ratings and votes
4. Return top recommendations
